## 🚀 Quick Start

### **1. Setup Platform:**
```bash
# Complete setup
npm run setup

# Manual setup
npm install
pip install -r requirements.txt
```

### **2. Start Platform:**
```bash
# Start professional server
npm start

# Launch automation
npm run automation

# Platform management
npm run platform-manager
```

### **3. Access Platform:**
- **Dashboard**: http://localhost:3000
- **Health Check**: http://localhost:3000/health
